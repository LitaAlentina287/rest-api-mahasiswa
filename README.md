# REST API Mahasiswa

API sederhana menggunakan Node.js dan Express.js.

## Cara Menjalankan
npm install
npm start

## Endpoint
- GET / → Home
- GET /students → Semua mahasiswa
- GET /students/:id → Detail mahasiswa
- POST /students → Tambah mahasiswa
- PUT /students/:id → Update mahasiswa
- DELETE /students/:id → Hapus mahasiswa
